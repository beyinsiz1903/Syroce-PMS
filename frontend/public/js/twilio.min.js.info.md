* **License:** Twilio Terms of Service (Proprietary, free for use connecting to Twilio services).

## Why Vendored Dynamic Loading?
The Twilio Voice SDK relies on Node.js global dependencies (such as `Buffer`, `EventEmitter`, and `process`) which are absent in modern browser runtimes compiled by Vite. Incorporating it as an npm dependency forces complex Vite bundler polyfills and increases bundle footprint. Loading the pre-compiled browser-compatible asset dynamically guarantees runtime safety and keeps initial bundle sizes lean.

## Update Procedure
1. Verify new releases on the Twilio Voice Client JS SDK Changelog.
2. Download the updated `twilio.min.js`.
3. Compute the SHA-256 checksum and verify it.
4. Replace `frontend/public/js/twilio.min.js` and update the metadata in this file.
5. Run Playwright E2E tests (`yarn e2e`) to verify full functionality.
