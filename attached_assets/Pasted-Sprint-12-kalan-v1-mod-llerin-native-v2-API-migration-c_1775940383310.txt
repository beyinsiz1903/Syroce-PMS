Neden:
Önceki backlog snapshot’larında refactoring başlığı altında açıkça şu kalem vardı:

Migrate v1_ modules to v2 API
Bu, channel tarafındaki büyük ürün işleri tamamlandıktan sonra en mantıklı sıradaki teknik iş olarak duruyor.
